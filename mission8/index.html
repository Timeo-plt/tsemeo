1.Somme

Écrire un programme qui demande à l'utilisateur de saisir deux nombres entiers et qui affiche la somme de ces deux nombres.

function somme () { var a = parseInt(prompt("Entrez un premier nombre :")); var b = parseInt(prompt("Entrez un deuxieme nombre :")); var resultat = a + b; alert(resultat); }

5.Pair ou Impair

Ecrire un programme qui demande à l'utilisateur de saisir un nombre et voir s'il est pair ou impair

function pairImpair () { var a = parseInt(prompt("Entrez un nombre :")); if (a % 2 === 0) { alert("Le nombre est pair"); } else { alert("Le nombre est impair"); }

6.Moyenne

Ecrire un programme qui demande à l'utilisateur de saisir 3 notes et qui affiche la moyenne de ces notes

function moyenne () { var a = parseInt(prompt("Entrez la premiere note :")); var b = parseInt(prompt("Entrez la deuxieme note :")); var c = parseInt(prompt("Entrez la troisieme note :")); var resultat = (a + b + c) / 3; alert(resultat); }

7.Calcul de l'aire d'un cercle

Ecrire un programme qui demande à l'utilisateur de saisir le rayon d'un cercle et qui affiche l'aire de ce cercle

function aireCercle () { var rayon = parseInt(prompt("Entrez le rayon du cercle :")); var resultat = Math.PI * rayon * rayon; alert(resultat); }

8.calcul du périmetre du cercle

Ecrire un programme qui demande à l'utilisateur de saisir le rayon d'un cercle et qui affiche le périmètre de ce cercle

function perimetreCercle () { var rayon = parseInt(prompt("Entrez le rayon du cercle :")); var resultat = 2 * Math.PI * rayon; alert(resultat); }

9.Calcul de l'aire d'un rectangle

function aireRectangle () { var longueur = parseInt(prompt("Entrez la longueur du rectangle :")); var largeur = parseInt(prompt("Entrez la largeur du rectangle :")); var resultat = longueur * largeur; alert(resultat); }

10.Donner les tables de multiplication d'un nombre donné

Ecrire un programme qui demande à l 'utilisateur de saisir un nombre puis de donner sa table de multiplication

function table () { var a = parseInt(prompt("Entrez un nombre :")); for (var i = 1; i <= 10; i++) { var resultat = a * i; alert(a + " x " + i + " = " + resultat); } }

11.Calcul de la factorielle d'un nombre

Ecrire un programme qui demande à l'utilisateur de saisir un nombre et qui affiche la factorielle de ce nombre

function factorielle () { var a = parseInt(prompt("Entrez un nombre :")); var resultat = 1; for (var i = 1; i <= a; i++) { resultat = resultat * i; } alert(resultat); }

14. Voir si le nombre est premier ou pas

Ecrire un programme qui demande à l'utilisateur de saisir un nombre et voir si il est premier ou pas

function premier () { var a = parseInt(prompt("Entrez un nombre :")); var premier = true; for (var i = 2; i < a; i++) { if (a % i === 0) { premier = false; } } if (premier) { alert("Le nombre est premier"); } else { alert("Le nombre n'est pas premier"); } }
